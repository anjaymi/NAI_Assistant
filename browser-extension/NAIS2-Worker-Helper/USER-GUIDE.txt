NAIS2 Worker Import 插件使用说明

适用浏览器：Chrome / Edge

一. 安装插件
1. 打开浏览器扩展管理页
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前这个文件夹：NAIS2-Worker-Helper

二. 使用步骤
1. 点击插件图标
2. 点击“打开 Cloudflare Workers 页面”
3. 登录 Cloudflare 账号
4. 点击“复制 Worker 代码模板”
5. 在 Cloudflare Worker 编辑器里把代码全量粘贴进去
6. 点击部署
7. 把部署后的 Worker 地址填回插件
8. 点击“测试 Worker 地址”
9. 显示可用后，再点击“一键导入到 NAI Assistant”

三. 地址填写说明
你可以填：
- https://your-worker.workers.dev

插件会自动补全成：
- https://your-worker.workers.dev/api/nai/subscription

四. 使用前提
1. 电脑已经安装 NAI Assistant 0.12.5 或更高版本
2. NAI Assistant 已注册 nais2:// 协议

五. 注意
插件不能替你自动注册 Cloudflare 账号、输入密码或跳过验证码。
它的作用是帮助你打开页面、复制代码、测试地址并导入到 NAI Assistant。
