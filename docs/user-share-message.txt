把这两个文件都下载好：

1. `NAI Assistant_0.12.5_x64-setup.exe`
2. `NAIS2-Worker-Helper.zip`

先安装 NAI Assistant，再把 `NAIS2-Worker-Helper.zip` 解压出来。

然后按下面做：

1. 打开 Chrome 或 Edge 扩展管理页
2. 开启“开发者模式”
3. 选择“加载已解压的扩展程序”
4. 选择解压后的 `NAIS2-Worker-Helper` 文件夹
5. 点击插件图标
6. 按插件里的提示操作：
   - 打开 Cloudflare Workers 页面
   - 复制 Worker 代码模板
   - 部署
   - 测试地址
   - 一键导入到 NAI Assistant

如果你只拿到了 `https://xxx.workers.dev` 这种地址也没关系，插件会自动补全。

如果你不会操作，就把插件弹窗截图发我，我一步一步带你做。
